# MonthWise Splitter – SaaS Edition

A web application that processes large text/chat files, splits them into month-wise segments, and returns a downloadable ZIP. Designed with a premium drag-and-drop UI and production-ready backend.

---

## 🚀 Features
- Drag & drop file upload
- Processes large TXT files efficiently
- Automatically generates month-wise text files
- Returns results as a ZIP
- Premium UI with responsive design

---

## 🛠️ Tech Stack
- **Backend:** Python, FastAPI
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render / Cloud
- **File Processing:** ZIP & filesystem automation

## 📂 Usage
1. Open the live URL
2. Drag & drop your TXT file
3. Click "Upload & Process"
4. Download the month-wise ZIP output
